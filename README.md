# EMS Portfolio Project

## 📦 Structure

### /store - Medical Equipment E-commerce
- Frontend: Product catalog, cart, checkout
- Backend: Product API, Cart API, Orders

### /admin - Employee Management System
- Frontend: Admin dashboard
- Backend: Employee CRUD, Authentication

## 🚀 Quick Start

### Store
cd store/frontend
# Open index.html in browser

### Admin Backend
cd admin/backend
npm install
npm run migrate
npm run dev
