# Portfolio (Site Root)

This folder is the deployable site root.

## Run locally

- Open `index.html` directly in a browser, or
- Serve this folder with any static server (recommended for PWA/manifest testing), for example:

```bash
cd portfolio
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Pages

- Portfolio: `index.html`
- EMS Store: `ems-store/index.html`
